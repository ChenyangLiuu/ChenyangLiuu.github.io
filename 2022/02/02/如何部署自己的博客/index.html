如何部署自己的博客

①下载相关工具 ——node.js

下载node.js https://nodejs.org/zh-cn/

②用npm或者cnpm安装相关工具

npm install -g hexo-cli

③新建blog文件夹

文件夹内hexo init

hexo n “文件名” 新建文章

hexo s 或 hexo server 预览

安装deployer：npm install –save hexo-deployer-git

④在github上创建仓库（注意仓库名字ChenyangLiuu.github.io）

⑤配置blog文件夹下的_config.yml

deploy:
type: git
repo: “仓库地址”
branch: master

⑥hexo d 部署到github仓库
